# LaTeX Setup Guide

This is a simple guide to setting up LaTeX to view and edit `.tex` files.

## 1. Install MiKTeX (The LaTeX Compiler)
MiKTeX is the underlying engine that processes LaTeX documents.
- **Download Link**: [https://miktex.org/download](https://miktex.org/download)
- **Instructions**: Download the installer for your operating system and follow the standard installation steps.

## 2. Install TeXworks (The Editor)
TeXworks is a program used to write, edit, and view `.tex` files.
- **Download Link**: [https://tug.org/texworks/#Getting_TeXworks](https://tug.org/texworks/#Getting_TeXworks)
- **Instructions**: Download the installer for your operating system and install it.

## 3. How to View and Edit `.tex` Files
1. After installing both MiKTeX and TeXworks, locate your `.tex` file in your file explorer.
2. Right-click the `.tex` file and select **Open with**.
3. Choose **TeXworks** from the list of programs.
4. Once open in TeXworks, you can edit the text.
5. To generate a viewable document (like a PDF), click the **Green Play Button** (Typeset) in the top-left corner of TeXworks to compile the file.
